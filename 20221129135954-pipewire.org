** Volume very low
   Use ~alsamixer~ tool and set the output gain to whatever.

   - See also https://wiki.archlinux.org/title/PipeWire#Low_volume
   - You can store the alsa config by using ~alsactl~

     #+begin_src shell
     sudo alsactl store
     #+end_src
