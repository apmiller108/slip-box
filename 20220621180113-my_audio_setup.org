* Mixer, Midi Controllers and FX
[[file:images/audio-setup_1.webp]]

[[https://mixxx.org/][Mixxx]] is used for playing tracks in [[https://manual.mixxx.org/1.11/en/chapters/configuration.html#audio-outputs][external mixer mode]]. Each deck's audio is
sent out the audio interface and into the Model 1.4 via DB25 connector. What's
nice about using the DB25 is that it leaves open the RCA connections allowing
for switching between audio sources (eg, audio from Mixxx and other gear)

Both the [[https://www.allen-heath.com/ahproducts/xonek2/][Xone:K2]] and the [[https://store.djtechtools.com/products/midi-fighter-twister][Midi Fighter Twister]] are used to control everything
in Mixxx and some stuff on the Beebo. See also my custom [[https://github.com/apmiller108/allen_and_heath_xonek2_mixxx_mapping][Xone:K2 controller
mapping]].

Audio recording is done on the Tascam Recorder. Booth out audio goes back into
the laptop so I can play around with creating beat aware visualizations using
things like [[https://github.com/projectM-visualizer/projectm][ProjectM]].

* Mixxx Midi Clock
[[file:images/audio-setup_3.webp]]

MIDI clock signal is generated by the [[https://github.com/apmiller108/mixxx_midi_clock][Mixxx MIDI clock]] and distributed to
software and devices over USB and MIDI DIN. This is fundamental to keeping
external gear like the drum machine in sync with tracks playing in Mixxx. It
allows me to set tempo shift beat phase on external gear (beat match)

* Drum machine, sample and synth
[[file:images/audio-setup_2.webp]]

I'm having a lot of fun incorporating a drum machine and sampler into the
setup. Right now I'm using a Roland T-8 and TE EP-133 K.O. II.

* Setup script
I use a script symlinked into a location in my path so I can just type a
single command in the terminal to start the programs and make the pipewire connections.

https://github.com/apmiller108/audio/blob/main/pipewire_mixxx_startup.sh
