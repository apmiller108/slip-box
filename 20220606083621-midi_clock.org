* Formula
** Pulse per ms rate
  1 minute = 60 seconds = 60,000 ms
  60,000 ms / ppq * bpm

  Example:

  #+begin_src ruby
    ppq = 24.0
    bpm = 120.0
    
    ppms = 60_000 / (ppq * bpm)
    "One pulse every #{ppms} ms"
  #+end_src

  #+RESULTS:
  : One pulse every 20.833333333333332 ms
