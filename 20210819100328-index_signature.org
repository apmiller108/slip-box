* Basic usage
  The name of the next (eg, ~id~, ~key~) is arbitrary.
  
  #+begin_src typescript
    interface Foo {
      [id: number]: string;
      [key: string]: string;
    }
  #+end_src

* Using a set of literals

  #+begin_src typescript
    type FooKey = 'a' | 'b' | 'c'

    interface Foo {
      [key in FooKey]: string
    }
  #+end_src
